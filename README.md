# Daily Report

一个通用的个人日报/周报生成 skill。通过 IMAP 读取邮箱，自动归纳邮件内容并生成结构化报告。

## 功能

- 通过 IMAP 读取任意邮箱（支持企业邮箱、个人邮箱等）
- 根据邮件内容自动推断行业结构和分类
- 生成日报和周报
- 支持自定义配置（IMAP 服务器、端口、存放目录等）

## 使用场景

当你说「日报」「周报」「今天做了什么」「工作汇总」「生成报告」时会自动调用此 skill。

## 配置

首次使用需要配置：
- 邮箱地址
- IMAP 服务器地址
- IMAP 端口（默认 993）
- 是否使用 SSL（默认 true）
- 邮箱密码或授权码

## 技术栈

- Python
- IMAP
- LLM（用于自动归纳和分类）
